Infrastructure / Helper Classes / Utilies 

This library contains Helper Classes & Utilities that are Platform independent
The library is supposed to be able to be shared between all .Net Framework Projects,
and therefor uses .Net Standart

Note:
.Net Standart is not an own framework but only a collection of API's that unitys diffrent Frameworks
It provides the smallest common nominator between the diffrent versions of .net framework
For further details see MSDN
